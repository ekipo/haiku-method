Spec notes for this wireframe:
  • All colors use var(--color-drift-detected-fg) and var(--color-drift-detected-bg) — canonical tokens per DESIGN-TOKENS.md §1.3.2. NOT deprecated --color-drift-bg/fg/stripe.
  • Container: role="status" and aria-live="polite" — announced on appearance without interrupting current focus.
  • Non-color icon (clock 🕐) alongside text — strip is distinguishable by shape and text, not color alone.
  • When strip disappears, empty aria-live="polite" region remains in DOM as visibility:hidden to prevent abrupt focus loss.
  • Strip is placed in DOM BEFORE the tab content — keyboard users encounter the alert before drilling into stage content.
  • Entry rows are informational only (non-interactive <div> elements with no focusable children).
  • Disclosure toggle has 44×44 minimum touch target via ::after pseudo-element.
  • Reduced-motion: enter/exit animation respects prefers-reduced-motion: reduce (global guard in index.css); custom keyframes must add explicit override.
  • Direction A passive-observer constraint: zero classification-trigger controls on this surface (per SPA-UI-SPECS.md §0).
  • No raw hex values. No raw Tailwind palette classes in semantic surfaces.
